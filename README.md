# Apsens CLI
 
a REPL style CLI to interact with an Aspens Market Stack

## Getting Started

Prerequisite: Ensure [rust](https://rustup.rs/) is installed on your system.

```bash
# copy the .env sample 
cp .env.sample .env

# after changing the values in your .env
source .env
```

## Usage

```bash
$ cargo run

# to see all available commands
cli-trader> --help 

# to end the session
cli-trader> quit
```
